Frequently Asked Questions

Apa itu LevviCode API?

LevviCode API adalah layanan REST API publik yang menyediakan berbagai endpoint untuk kebutuhan bot WhatsApp, downloader, search, dan tools lainnya, gratis tanpa API key.

Apakah perlu registrasi?

Tidak. Semua endpoint dapat digunakan langsung tanpa registrasi atau API key.

Apakah ada rate limiting?

Ya, untuk menjaga kestabilan server, setiap IP dibatasi 100 request per 15 menit. Endpoint tertentu mungkin memiliki batasan lebih ketat.
